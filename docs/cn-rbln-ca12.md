# 1. 介绍

Rebellion的RBLN-CA12 加速器卡是一款FHFL、单插槽10.5英寸的PCI Express Gen5，搭载了专为深度学习推理而设计的系统级芯片（System-on-Chip, SoC）——ATOM™。 它采用被动冷却方式 。
RBLN-CA12支持基于CNN、RNN、Transformer等多种神经网络模型的AI推理任务，加速边缘计算、 高性能计算（High Performance Computing， HPC）领域的推理运算。它可用于云数据中心或人工智能服务。
